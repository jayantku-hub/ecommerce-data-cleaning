#  E-Commerce Data Cleaning Project

##  Overview
This project demonstrates data cleaning using Python (Pandas) on a messy e-commerce dataset containing 10,000+ rows.

##  Cleaning Steps
- Removed duplicates
- Standardized column names
- Handled missing values
- Fixed data types
- Validated emails
- Standardized date formats
- Recalculated totals

##  Tools Used
- Python
- Pandas
- VS Code

##  Files
- generate_dataset.py — Creates raw dataset
- clean_data.py — Cleaning script
- unclean_orders_10000.csv — Raw data
- cleaned_orders.csv — Cleaned output
